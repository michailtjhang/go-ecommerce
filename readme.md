# go-ecommerce

command in terminal
```bash
go mod init go-ecommerce
```

install package

```bash
go get -u github.com/gin-gonic/gin
```
```bash
go get -u github.com/jinzhu/gorm
```
```bash
go get -u github.com/jinzhu/gorm/dialects/mysql
```
```bash
go get github.com/dgrijalva/jwt-go
```

command run Application 
```bash
go run main.go
```
